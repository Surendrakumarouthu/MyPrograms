JavaScript Object Methods

1. Object.keys()

Returns an array of a given object's own enumerable property names.

Example: 
const user = { name: "John", age: 30, role: "Developer" };
console.log(Object.keys(user)); 
// Output: ["name", "age", "role"]

2. Object.values()

Returns an array of a given object's own enumerable property values.

Example: 
const user = { name: "John", age: 30, role: "Developer" };
console.log(Object.values(user)); 
// Output: ["John", 30, "Developer"]

3. Object.entries()

Returns an array of a given object's own enumerable string-keyed property [key, value] pairs.

Example: 
const user = { name: "John", age: 30, role: "Developer" };
console.log(Object.entries(user)); 
// Output: [["name", "John"], ["age", 30], ["role", "Developer"]]

4. Object.assign()

Copies all enumerable own properties from one or more source objects to a target object.

Example: 
const target = { a: 1, b: 2 };
const source = { b: 4, c: 5 };
const result = Object.assign(target, source);
console.log(result); 
// Output: { a: 1, b: 4, c: 5 }

5. Object.freeze()

Freezes an object, meaning no changes can be made to its properties.

Example: 
const obj = { name: "John" };
Object.freeze(obj);
obj.name = "Jane";  // This will not change the name
console.log(obj.name); 
// Output: "John"

6. Object.seal()

Seals an object, preventing new properties from being added but allowing changes to existing properties.

Example: 
const obj = { name: "John" };
Object.seal(obj);
obj.age = 25;  // Cannot add new property
obj.name = "Jane";  // Can modify existing property
console.log(obj); 
// Output: { name: "Jane" }

7. Object.create()

Creates a new object with the specified prototype object and properties.

Example: 
const person = { greet: function() { console.log("Hello"); } };
const john = Object.create(person);
john.name = "John";
john.greet(); 
// Output: "Hello"

JavaScript Object Function Methods

1. Function.prototype.call()

Calls a function with a given `this` value and arguments provided individually.

Example: 
function greet() {
    console.log(`Hello, ${this.name}`);
}
const user = { name: "John" };
greet.call(user); 
// Output: "Hello, John"

2. Function.prototype.apply()

Similar to `call()`, but arguments are provided as an array.

Example: 
function greet(greeting, punctuation) {
    console.log(`${greeting}, ${this.name}${punctuation}`);
}
const user = { name: "John" };
greet.apply(user, ["Hi", "!"]); 
// Output: "Hi, John!"

3. Function.prototype.bind()

Creates a new function that, when called, has its `this` keyword set to the provided value.

Example: 
function greet() {
    console.log(`Hello, ${this.name}`);
}
const user = { name: "John" };
const greetUser = greet.bind(user);
greetUser(); 
// Output: "Hello, John"

4. Function.prototype.toString()

Returns a string representing the source code of the function.

Example: 
function greet() {
    console.log("Hello");
}
console.log(greet.toString()); 
// Output: function greet() { console.log("Hello"); }

5. Function.prototype.constructor

Returns a reference to the constructor function that created the instance's prototype.

Example: 
function Person(name) {
    this.name = name;
}
const john = new Person("John");
console.log(john.constructor); 
// Output: function Person(name) { this.name = name; }

Difference Between Object and Function Methods

Object Methods: Methods such as Object.keys(), Object.values(), and Object.assign() deal with object properties and their manipulation. These methods are related to managing the data contained within objects.

Function Methods: Methods like call(), apply(), and bind() are used to control the execution context (the value of `this`) of functions. These methods are useful when you need to explicitly define the object that a function should be bound to or executed with.
