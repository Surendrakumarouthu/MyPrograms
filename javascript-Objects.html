JavaScript Objects

In JavaScript, an object is a collection of properties, where each property has a key (string) and a value. Objects are fundamental in JavaScript, used to store various keyed collections and more complex entities.

1. Creating an Object

Objects can be created in two main ways:

1.1 Using Object Literals

const person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    isEmployed: true
};
console.log(person.firstName); // Output: John

1.2 Using the Object Constructor

const person = new Object();
person.firstName = "John";
person.lastName = "Doe";
person.age = 30;
person.isEmployed = true;

console.log(person.firstName); // Output: John

2. Accessing Object Properties

Properties in an object can be accessed in two ways:

2.1 Dot Notation

console.log(person.firstName); // Output: John

2.2 Bracket Notation

console.log(person["lastName"]); // Output: Doe

3. Adding, Modifying, and Deleting Properties

3.1 Adding a Property

person.country = "USA";
console.log(person.country); // Output: USA

3.2 Modifying a Property

person.age = 35;
console.log(person.age); // Output: 35

3.3 Deleting a Property

delete person.isEmployed;
console.log(person.isEmployed); // Output: undefined

4. Methods in Objects

An object can also have functions as its properties, known as methods.

const person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    fullName: function() {
        return this.firstName + " " + this.lastName;
    }
};

console.log(person.fullName()); // Output: John Doe

5. Object.keys() and Object.values()

5.1 Object.keys()

Returns an array of a given object's own enumerable property names.

const keys = Object.keys(person);
console.log(keys); // Output: ["firstName", "lastName", "age", "fullName"]

5.2 Object.values()

Returns an array of a given object's own enumerable property values.

const values = Object.values(person);
console.log(values); // Output: ["John", "Doe", 30, function() {...}]

6. Object Destructuring

Object destructuring is a concise way to extract values from an object.

const { firstName, age } = person;
console.log(firstName); // Output: John
console.log(age); // Output: 30

7. Nested Objects

Objects can also be nested, meaning one object can be a property of another object.

const company = {
    name: "TechCorp",
    address: {
        street: "123 Main St",
        city: "San Francisco",
        state: "CA"
    }
};

console.log(company.address.city); // Output: San Francisco

8. Checking if a Property Exists

To check if a property exists in an object, you can use the in operator or hasOwnProperty().

console.log("firstName" in person); // Output: true
console.log(person.hasOwnProperty("age")); // Output: true

9. Looping through Object Properties

You can use the for...in loop to iterate over all the enumerable properties of an object.

for (let key in person) {
    console.log(key + ": " + person[key]);
}
// Output: 
// firstName: John
// lastName: Doe
// age: 30
