ES6 Array Methods

1. forEach()

Executes a provided function once for each array element. It does not return anything (undefined), mainly used for iteration or side-effects like logging.

const numbers = [1, 2, 3, 4];
numbers.forEach((num) => {
    console.log(num);
});
// Output: 1 2 3 4

2. map()

Creates a new array by applying a function to each element of the original array. It doesn't modify the original array.

const numbers = [1, 2, 3, 4];
const squared = numbers.map((num) => num * num);
console.log(squared); // Output: [1, 4, 9, 16]

3. filter()

Creates a new array with all elements that pass a test implemented by the provided function. If an element doesn't satisfy the condition, it is filtered out.

const numbers = [1, 2, 3, 4, 5, 6];
const evenNumbers = numbers.filter((num) => num % 2 === 0);
console.log(evenNumbers); // Output: [2, 4, 6]

4. findIndex()

Returns the index of the first element in the array that satisfies the provided testing function. If no element matches, it returns -1.

const numbers = [10, 20, 30, 40];
const index = numbers.findIndex((num) => num === 30);
console.log(index); // Output: 2

5. reduce()

Applies a function against an accumulator and each array element (from left to right) to reduce it to a single value.

const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
console.log(sum); // Output: 10

6. find()

Returns the first element in the array that satisfies the provided testing function. If no element matches, it returns undefined.

const numbers = [10, 20, 30, 40];
const found = numbers.find((num) => num > 25);
console.log(found); // Output: 30

7. some()

Returns true if at least one element in the array satisfies the condition in the callback function, otherwise returns false.

const numbers = [1, 2, 3, 4];
const hasEven = numbers.some((num) => num % 2 === 0);
console.log(hasEven); // Output: true

8. every()

Returns true if all elements in the array pass the test implemented by the provided function.

const numbers = [2, 4, 6, 8];
const allEven = numbers.every((num) => num % 2 === 0);
console.log(allEven); // Output: true

9. sort()

Sorts the elements of an array in place (modifies the original array). By default, it converts elements to strings and compares them lexicographically.

const numbers = [30, 5, 100, 20];
const sorted = numbers.sort((a, b) => a - b);
console.log(sorted); // Output: [5, 20, 30, 100]

10. concat()

Merges two or more arrays into a new array, without changing the existing arrays.

const arr1 = [1, 2];
const arr2 = [3, 4];
const combined = arr1.concat(arr2);
console.log(combined); // Output: [1, 2, 3, 4]
